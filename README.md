# Turtle Invaders

A Space Invaders clone made using `turtle`, following the Real Python tutorial [Build a Python Turtle Game: Space Invaders Clone](./notes.md).

![](./screenshot.png)

## Installation

All you need is a working Python 3.9+.

## Usage

```sh
python turtle_invaders.py
```
